常見問題

有問題？我們有答案

這個擴充功能是否免費？ +

是的！這個擴充功能完全免費，包含所有核心功能如無限收藏、搜尋、筆記、匯出備份等。

我的資料安全嗎？ +

絕對安全！所有資料完全儲存在本地瀏覽器中，不會上傳到任何伺服器，確保您的資料隱私。

支援哪些瀏覽器？ +

目前支援所有基於 Chromium 的瀏覽器，包括 Chrome、Edge、Brave 等。Firefox 版本正在開發中。

可以匯入現有的書籤嗎？ +

可以！支援 JSON 格式匯入。您可以從其他地方匯出的 JSON 檔案直接匯入，輕鬆遷移現有收藏。
